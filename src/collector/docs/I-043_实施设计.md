# I-043 实施设计：探针映射 AI 助手

## 目标
新增字段映射建议接口，基于样例自动给出映射候选、总体置信度、未知字段与人工确认清单，降低探针接入成本。

## 新增接口
- `POST /api/v1/normalization/suggest-mapping`

## 请求示例
```json
{
  "probe_type": "sidecar_probe",
  "raw_event_type": "net",
  "sample": {
    "source": "SAT-INCL-001",
    "target": "SAT-INCL-091",
    "latency": 185,
    "loss": 0.02,
    "jitter": 12
  },
  "use_ai": true
}
```

## 响应核心字段
- `mapping[]`: 字段映射建议（目标字段、原始字段、置信度、依据）
- `confidence`: 总体置信度（score/level）
- `unknown_fields[]`: 未命中的原始字段
- `manual_todo[]`: 人工确认项
- `ai_note`: AI 补充建议（可选）

## 风险控制
- AI 仅辅助说明，不自动落库生效。
- 若 AI 不可用，规则建议仍可用。
